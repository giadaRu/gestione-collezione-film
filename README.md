# Collezione personale di film
Il progetto è stato sviluppato per il Corso di Ingegneria del Software 2024-2025.  
Esso è un'applicazione Java per la gestione locale e autonoma di una collezione personale di film.   
Il sistema permette agli utenti di organizzare, consultare e aggiornare la propria raccolta cinematografica, enfatizzando la manutenibilità e l'estensibilità attraverso l'uso consapevole dei Design Pattern.

## How to use it
1. Clona il repository: https://github.com/giadaRu/gestione-videoteca.git
2. Apri il progetto nell'IDE
3. Naviga nel package main.java.it.unical.videoteca
4. Fai clic destro sulla classe `MainVideoteca` e seleziona **Run**
5. Interagisci con la tua collezione di film tramite il **menu testuale** che compare nel terminale.

## Output
Il sistema genera e gestisce un archivio di dati completamente locale: i dati dei film e delle raccolte vengono salvati nei file `videoteca_data.csv` e `raccolte_data.csv`. Questi file sono gestiti esclusivamente dai Repository e non dovrebbero essere modificati manualmente.
